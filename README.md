# 雅思背词小程序 MVP

使用 Taro + React 开发的看故事背雅思单词小程序。

## 功能特性

- 📚 **看故事背单词**：通过故事场景沉浸式学习雅思词汇
- 🎧 **音频播放**：配合音频加深记忆
- 🔤 **中英对照**：一键切换显示/隐藏中文释义
- 📝 **重点词汇**：每个章节包含重点单词及例句

## 项目结构

```
src/
├── app.ts                    # 应用入口
├── app.config.ts             # 应用配置
├── app.less                  # 全局样式
├── assets/                   # 静态资源
│   └── icons/               # Tab栏图标
├── data/
│   └── mock.ts              # Mock数据
└── pages/
    ├── index/               # 首页（学英语）
    ├── mine/                # 我的页面
    ├── section-list/        # 章节列表页
    └── study/               # 学习页面
```

## 快速开始

### 安装依赖

```bash
npm install
```

### 开发模式

```bash
# 微信小程序
npm run dev:weapp

# H5
npm run dev:h5
```

### 构建生产版本

```bash
# 微信小程序
npm run build:weapp

# H5
npm run build:h5
```

## 页面说明

### 1. 首页（学英语）
- 展示所有Part列表
- 每个Part包含多个Section
- 点击Section进入学习页面

### 2. 学习页面
- 背景图片展示
- 英文段落内容
- 音频播放控制（播放/暂停、进度条、快进/快退）
- 左上角功能按钮：
  - 切换显示/隐藏中文释义
  - 查看本节重点词汇

### 3. 我的页面
- 用户信息展示
- 学习统计数据
- 功能菜单入口

## 技术栈

- **Taro 3.x**：跨端开发框架
- **React 18**：UI 框架
- **TypeScript**：类型安全
- **Less**：CSS 预处理器

## 注意事项

1. 当前使用 Mock 数据，实际项目需要对接后端 API
2. 音频播放功能在模拟器中为模拟进度，真机需要配置真实音频源
3. 背景图片使用 Unsplash 示例图片，正式环境需替换
